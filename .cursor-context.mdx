# Printomate - Automated Etsy Poster Generator

## Project Overview

This is a Next.js (TypeScript) web application that generates customizable posters for Etsy sellers. The system allows users to input custom content (names, quotes, dates) and automatically applies it to multiple pre-designed poster layouts in various sizes, outputting a downloadable ZIP archive.

## Tech Stack

- **Frontend**: Next.js 14+ (App Router), TypeScript, Tailwind CSS
- **Backend**: Next.js API Routes
- **Image Processing**: node-canvas
- **File Handling**: JSZip
- **State Management**: React Context/Hooks
- **Styling**: Tailwind CSS
